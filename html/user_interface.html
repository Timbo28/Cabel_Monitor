User Interface

Describes how to operate the user interface.

Hint

Once power is applied to the board, all controls are briefly described.
Pressing the black button triggers a system restart.
Pressing the blue button turns the buzzer on or off, when the buzzer is switched on, a red LED lights up constantly.
The buzzer is switched off by default and when turned on, only emits a sound when a cable is nearby.
The measurement can be started by pressing the lower two controls "Average Measurement" or "Singel Measurement".
As soon as the measurements are running, the visual feedback can be changed by tapping in the center of the screen.

Graphic

Once a measurement type is selected, the orientanion to the cable is visually displayed.
The red circle symbolizes the cable and the red line symbolizes the distance to the device.
At the bottom right, the calculated distance to the cable is displayed.
At the bottom left, the calculated current of the cable is displayed if the device is close enough.
The angle can be read visually, with the markings spaced at 15-degree intervals.

Values

Pressing the center of the screen changes the display. This causes all calculated values to be output as a number.
This also makes the x and y axes visible, which are described in more detail in calculations.c.
Otherwise, the values correspond to the graphical values.
