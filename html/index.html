Cable Monitor

Firmware

This is the firmware code for the Electronics-Project Modules ET.PM3: Cable-Monitor
in the second year of the electronics engineering curriculum at the School of Engineering of the Zurich University of Applied Sciences.

The firmware code is written for the evaluation board 32F429IDISCOVERY (Discovery kit with STM32F429ZI MCU).

measuring.c contains functions to process the ADC values from the pads and Hall Effect sensors.
All necessary calculations for position determination can be found in calculations.c
The file menu.c allows the readout of the calculated values.
In buzzer.c are functions to control the buzzer.
pushbutton.c handles external interrupts.
main.c puts everything together and processes the input from the user.
